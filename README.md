# 📘 SQL do Básico ao Profissional

Este repositório representa a conclusão do curso **SQL do Básico ao Profissional**, onde desenvolvi conhecimentos sólidos em bancos de dados relacionais e consultas SQL aplicadas ao mercado de tecnologia e dados.

Durante o curso, aprendi desde os conceitos fundamentais até técnicas utilizadas no ambiente profissional, com foco em prática, organização de código e resolução de problemas reais utilizando SQL.

---

# 🚀 Sobre o Curso

O curso foi desenvolvido para ensinar SQL de forma prática e profissional, abordando desde os primeiros conceitos de banco de dados até a construção de projetos completos para portfólio.

Ao longo da formação, foram realizados exercícios práticos, desafios, quizzes e desenvolvimento de projetos aplicados ao cenário real de tecnologia e análise de dados.

---

# 📚 Conteúdos Aprendidos

## 🗄️ Fundamentos de Banco de Dados

* Conceitos de bancos de dados relacionais
* Modelagem de dados
* Estrutura de tabelas
* Relacionamentos entre tabelas
* Chaves primárias e estrangeiras

## 💻 SQL na Prática

* Criação de bancos de dados
* Criação e manipulação de tabelas
* Inserção, atualização e remoção de dados
* Consultas SQL
* Filtros e ordenações
* Joins
* Funções agregadas
* Subqueries
* Views
* Boas práticas de escrita SQL

## 📊 Resolução de Problemas Reais

* Consultas utilizadas no mercado
* Manipulação de grandes volumes de dados
* Organização e otimização de consultas
* Estruturação de projetos SQL

## 🏗️ Projeto Final

Desenvolvimento de um projeto completo utilizando SQL, aplicando os principais conceitos aprendidos durante o curso.
